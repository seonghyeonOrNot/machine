각 저항의 전압 강하
전압 강하 = I × R — 저항이 클수록 전압을 더 많이 가져갑니다
각 저항의 소비 전력
소비 전력 = I² × R — 저항이 클수록 더 많은 전력을 소비합니다
① 합성저항
RT = R1 + R2 + ··· + Rn
직렬에서는 모든 저항을 단순히 더합니다.
② 전류 (일정)
IT = I1 = I2 = ··· = In
경로가 하나뿐이므로 모든 소자에 동일한 전류가 흐릅니다.
③ 전압 (분배)
VT = V1 + V2 + ··· + Vn
인가전압 = 각 저항의 전압강하 합. 저항 비율대로 전압이 나뉩니다.
④ 옴의 법칙 적용
V = IR  →  I = V/RT  →  Vk = I × Rk
합성저항으로 전류를 구하고, 각 저항에 개별 전압을 계산합니다.
⑤ 전압 분배 비율
R1 : R2 : ··· = V1 : V2 : ···
전류가 같으므로 전압강하는 저항에 정비례합니다.
